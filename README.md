# [ICLR 2026] Generalizable Heuristic Generation Through LLMs with Meta-Optimization

This repository contains the official implementation of **MoH (Meta-optimization of Heuristics)**, a novel framework for generating generalizable heuristics through Large Language Models with meta-optimization techniques.

## Paper

**Generalizable Heuristic Generation Through LLMs with Meta-Optimization** Accepted at ICLR 2026

<!-- ## Overview

MoH addresses the challenge of creating generalizable heuristics that can adapt across different problem domains and instances. Traditional heuristic methods are often domain-specific and require extensive manual tuning. Our approach leverages the power of Large Language Models (LLMs) combined with meta-optimization to automatically discover and optimize heuristics that generalize well.

### Key Features

- **LLM-driven Heuristic Generation**: Utilizes Large Language Models to generate novel heuristic strategies
- **Meta-optimization Framework**: Implements a meta-optimization loop to refine and improve heuristics over time
- **Cross-domain Generalization**: Demonstrates strong generalization capabilities across different problem domains
- **Automated Discovery**: Automates the process of discovering effective heuristics without manual design

### Methodology

The MoH framework consists of three main components:

1. **Heuristic Generator**: An LLM-based component that proposes new heuristic strategies
2. **Evaluator**: Assesses the performance of generated heuristics on training tasks
3. **Meta-Optimizer**: Updates the generator based on evaluation feedback to improve future heuristic proposals
